###  如何快速使用  

0.安装好nodejs环境

1.下载NaughtyNode文件  

2.window系统:双击文件内的start-up.bat  

3.linux系统切换到该目录然后:node start  

4.80端口访问 / 修改conf文件下的config.yml可修改大部分配置  

5.后台超级管理员:admin/123
